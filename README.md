# Exercícios de Linguagem C

Repositório com os exercícios de programação em **C** passados em aula, organizados por dia de aula.

---

## Estrutura do repositório

```
/
├── dia-01/         ← Estruturas de dados, entrada e saída
├── dia-02/         ← Estrutura condicional
├── dia-03/         ← Estrutura de seleção
├── dia-04/         ← Estrutura de repetição
├── dia-05/         ← Vetores e matrizes
├── dia-06/         ← Funções
└── README.md
```

Cada pasta corresponde a um dia de aula e contém os arquivos `.cpp` dos exercícios propostos pelo professor.

---

## Como compilar e executar

Os exercícios são desenvolvidos no **Embarcadero Dev-C++** (Windows).

**Passos para abrir e executar:**

1. Abra o Dev-C++
2. `File` → `Open` → selecione o arquivo `.cpp` desejado
3. Pressione **F11** para compilar e executar, ou use o menu `Execute` → `Compile & Run`

**Atalhos úteis:**

| Ação              | Atalho |
|-------------------|--------|
| Compilar e rodar  | F11    |
| Apenas compilar   | F9     |
| Criar novo arquivo| Ctrl+N |
